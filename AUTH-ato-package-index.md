## Document Currency Schedule

All ATO package documents are subject to the following review schedule:

| Document Type | Review Frequency | Review Trigger |
|---|---|---|
| SSP (all sections) | Annual | Annual review or significant system change |
| Risk Register | Annual | Annual review; new threat identification |
| POA&M Tracker | Monthly | Monthly ISSO review cycle |
| SAR / Findings | Annual | Annual SCA assessment |
| ATO Package Index | On change | When any document is added or revised |
| Authorization Recommendation | Annual | Annual reauthorization review |
| Roles and Responsibilities | Annual | Personnel changes; organisational restructure |
| Contingency Plan | Annual | Annual review; architecture change |
| BAA Register | Annual | Vendor changes; new integrations |
